Манипуляции с DOM: Учебник на примере семейного древа

Практическое руководство по взаимодействию с DOM с помощью JavaScript.

Основные концепции

1. Выбор элементов

Чтобы что-то изменить, сначала нужно это выбрать. Мы присвоили каждому человеку `id`, чтобы упростить задачу. Используйте `document.getElementById('elementId')`.

const adamElement = document.getElementById('adam');

2. Обход (Перемещение между элементами)

  • .parentElement: Переходит к прямому родительскому контейнеру.
  • .children: Получает список всех дочерних элементов.
  • .nextElementSibling: Переходит к следующему соседнему элементу.
  • .previousElementSibling: Переходит к предыдущему соседнему элементу.

Примечание: Мы используем `ElementSibling` вместо `Sibling`, чтобы игнорировать текстовые узлы с пробелами, которые могут создавать трудности.

3. Манипуляция (Изменение древа)

  • document.createElement('div'): Создаёт новый элемент (но пока не размещает его).
  • element.appendChild(newElement): Добавляет `newElement` как дочерний элемент для `element`.
  • element.remove(): Удаляет элемент со страницы.
